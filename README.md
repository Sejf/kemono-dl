# kemono-dl

# Usage:
Beta and the normal site url are both valid:

```bash
kemono_dl.py -i http://kemono.party/user/[subscription]/user/[userid]
```
or
```bash
kemono_dl.py -i input.txt
```

# Requeriments:
```
urllib3
beautifulsoup4
```
